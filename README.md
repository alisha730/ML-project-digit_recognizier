# Digit Recognizer Desktop App

This is a simple desktop application for handwritten digit recognition.

## Features
- Draw digits (0â€“9) using mouse
- Predict digit using trained ML model
- Python Tkinter GUI
- TensorFlow-based digit classifier

## How to Run
1. Install Python 3
2. Install dependencies:
   pip install tensorflow pillow numpy
3. Train model (run once):
   python train.py
4. Run app:
   python app.py
   OR
   double-click run_app.bat
